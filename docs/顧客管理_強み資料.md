# 顧客管理シート ── 強み資料

## 概要
治療院専用の顧客管理システム（CRM）。患者情報・来院履歴・施術記録・売上データを一元管理。ダッシュボードで総患者数・月間施術数・本日の売上をリアルタイム表示。CSV取込による既存データの移行にも対応。LTV・リピート率・ROAS・エリア分析など多角的な経営分析機能を搭載。

## URL
https://customer-mgmt.vercel.app

## 月額料金
4,980円（買い切り: 49,800円）

## ターゲットの悩み
1. 紙カルテの管理が限界。患者が増えると検索・管理が追いつかない
2. 高額なCRMシステムは機能が多すぎて使いこなせない
3. 売上やリピート率の分析を手作業でやっていて時間がかかる
4. LTV（顧客生涯価値）を把握できていない
5. 新規とリピートの比率が見えず、集客戦略が立てられない
6. CSV取込に対応していないツールが多く、データ移行が面倒

## 競合3カテゴリ
1. 治療院専用CRM（COCKPIT等）: 月額7,480〜12,980円。高機能だが高額で小規模院には負担
2. 汎用CRM（Salesforce、HubSpot等）: 月額数万円〜。治療院に特化していないため設定が複雑
3. 紙カルテ+Excel管理: 0円。分析不可、検索に時間がかかる

## このアプリの立ち位置
「小規模治療院でも使える、治療院専用の経営分析CRM」。大手CRMの半額以下で、LTV・リピート率・ROAS・時間帯別稼働率まで自動分析。

## 5つの強み
1. 治療院専用ダッシュボード: 総患者数・月間施術数・本日の売上を一画面で即確認。クイックアクションで新規患者・施術記録・一括入力がワンタップ
2. 多角的経営分析: LTV分析、リピート率推移、新規/既存比率、時間帯別稼働率、エリア別LTV、広告ROAS分析まで標準搭載。経営判断をデータで支援
3. CSV一括取込: 既存の紙カルテやExcelデータをCSVで一括インポート。データ移行の手間を最小化
4. 施術伝票管理: 来院ごとに施術内容・金額を記録。売上集計が自動化され、確定申告にも活用可能
5. マルチテナント対応: 院ごとに完全にデータ分離。Supabaseクラウド保存でどこからでもアクセス可能
